1) cd 
    cd - change directory. Changes current working directory to whatever you want
    Examples:
        cd .. - move to the parent dir
        cd /path/to/directory - move to the desired dir 

    You can combine path with .. like: cd ../../Desktop

2) pwd
    pwd - print working directory
        pwd: /some/directory

3) ls
    ls - list contents. Lists files in given directory 
    Examples:
        ls - list all files in current working dir
        ls -l - list all files in current working dir with permissions, user/group, size (in bytes), date and time of modification
        ls -a - list all files in current directory, including files with names that has . at the beginning (hidden files)
        ls -lh - list all files in current working dir with permissions, user/group, size (in B/KB/MB/GB/TB/... instead of bytes), date and time of modification
        ls -lhai - list all files in current working dir with permissions, user/group, size (in B/KB/MB/GB/TB/... instead of bytes), date and time of modification and inode number

    ls can also show info about of specific file/directory:
        ls -lh /path/to/file - show info about set file
        ls -lhd /path/to/directory - show info about set directory (instead of it's contents)

    You can combine it with shell's masks:
        ls -lha *.c - shows all files with .c extension (bash replaces * with all existing files in set directory)
        ls -lha file?.txt - shows filea.txt fileb.txt fileX.txt (bash tries each char instead of ?)
        ls -lha file[123].txt - same as ls file?.txt but bash tries each char in [] brackets
